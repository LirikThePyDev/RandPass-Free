# RandPass
RandPass Free is a lightweight Windows password generator. Create strong, random passwords offline with letters, numbers, and symbols. Easy-to-use, fully local, no passwords are stored. Personal use only; redistribution prohibited unless permitted.
Requirements:
- Python 3.x installed on Windows
- Basic knowledge of running Python scripts

How to Run
- Clone or download the repository:
git clone https://github.com/LirikThePyDev/RandPass.git
- Navigate to the folder:
cd RandPass
- Run the app:
python randpass.py
- Follow the on-screen instructions:
Press 1 to generate a password
 and 0 to exit

Notes
- Passwords are generated locally; no passwords are stored
- Minimum password length is enforced for security
- Use this software for personal, non-commercial purposes only
